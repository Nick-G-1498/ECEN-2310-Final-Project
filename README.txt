Both Scripts will require that the Deep Learning Toolbox and Machine Learning Toolbox being downloaded and installed


Hot Dog / Not Hot Dog Script
  AlexNet pretrained neural network needs to be downloaded and installed before hand. 
  It will also need some toolbox in order that you can use the webcam
  If you do not have the required support packages installed, then the software provides a download link.


Handwritten Digit Classifier 
  Training and Testing data for the handwritten digits needs to be downloaded and saved into a folder called 'MNISTData' in your directory
    http://yann.lecun.com/exdb/mnist/
  
  prepData.m is called by the main file to prepare the data for the main script so there is no need to unzip the '.gx' files
